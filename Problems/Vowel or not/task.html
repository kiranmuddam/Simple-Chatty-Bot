Vowel or not Open on JetBrains Academy

Implement a method to check whether the given English letter is a vowel or not. The input may be in any case.

Do not consider the letter 'y' as a vowel.

Examples:

  1. isVowel('a') should be true
  2. isVowel('A') should be true
  3. isVowel('b') should be false
